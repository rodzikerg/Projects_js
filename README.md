* ___isPerfect___  
 * 1.1 открываем файл, в появившейся странице открываем консоль(F12)
  *  1.2 в консоли вводим функцию isPerfect(а) где а-число 
* тестовые данные с платформы Hexlet.io
* isPerfect(6); // true
* isPerfect(7); // false
